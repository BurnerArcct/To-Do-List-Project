# To-Do List App

This is a simple web application for managing a to-do list. 

## Features
- Add tasks
- View tasks
- Simple and clean UI

## How to Use
1. Enter a task in the input field.
2. Click the "Add" button to add the task to the list.
3. View your tasks in the list below.

## Technologies Used
- HTML
- CSS
- JavaScript
